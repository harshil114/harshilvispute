How JavaScript Works

Everything inside the JavaScript happens in the Global Execution Context. Which is also known as GEC. Here is question that what is the Global Execution Context?

An answer for this is very simple that in the JavaScript engine GSE is like a big box or container where code is executed.

GSE contains 2 Components :

  1. Memory Component (Variable Environment)
  2. Code Component (Thread of Execution)

In the Memory Component it stores variable and function name as a key value pair and in the Code Component it stores all the code, calculation and every rest of code.
